# server-x-static

A simple static server project.

这是一个简单的静态服务器项目。

## Installation (安装)

```bash
npm install
```

## Usage (使用)

```bash
npm start
```

## Development (开发)

```bash
npm run dev
```

## 如何添加新的静态网站

1.  在 `dynamics` 文件夹下创建一个新的项目文件夹，例如 `my-new-project`。
2.  将您的静态网站文件（`index.html`, `style.css`, etc.）放入 `my-new-project` 文件夹中。
3.  启动服务器后，您可以通过访问 `http://localhost:3000/my-new-project` 来查看您的新网站。
